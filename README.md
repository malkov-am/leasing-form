# Калькулятор лизинга

Форма для расчета стоимости автомобиля в лизинг. Ползунки с кастомизированным дизайном, кроссбраузерная верстка.

- [Ссылка на проект](https://leasing-form.web.app)

## Применены следующие технологии:

- React.js
- Saas
- Адаптивная верстка
